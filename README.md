## [HTML] a linguagem de marcação html, ou, " Hiper text marckup language", serve para mudar certos elementos os quais você escolhe, eles serão alterados por tags, cada tag pode ter propriedades as quais vão alterar algo na parte marcada que você quer alterar. Essas propriedades precisam ter um valor, que muda dependendo da propriedade que você está usando.


## Diferença entre JavaScript e TypeScript
JavaScript é mais simples e não precisa definir tipo de variável. TypeScript é baseado no JavaScript, mas usa tipagem, ajudando a evitar erros.

## Classe
É como um molde que define como os objetos vão ser e o que eles podem fazer.

## Objeto
É algo criado a partir da classe. Ele tem as características e ações definidas nela.

## Atributo
São as informações de um objeto, como nome, idade ou cor.

## Método
São as ações que um objeto pode fazer, como falar ou calcular algo.

## Getters x Setters
Getters são usados para acessar o valor de um atributo. Setters são usados para um valor, geralmente com algum controle para evitar erros.

## Construtor
É um método que roda quando o objeto é criado, usado para iniciar valores.

## Herança
Permite que uma classe aproveite coisas de outra, como atributos e métodos.

## Encapsulamento
É esconder os dados e só permitir acesso de forma controlada. Public disponível em qualquer lugar do código. Private só dentro da classe. Protected com hierarquia.

## Polimorfismo
É quando um mesmo método pode fazer coisas diferentes dependendo do objeto que está usando ele. Isso permite reutilizar o mesmo nome com comportamentos diferentes.
